Concept 2: Hero Image Tiles

Strategy: Visual-first approach with large image areas for each case study. Featured tile takes 2/3 width to establish hierarchy. Grid system scales elegantly to any number of case studies.

Strengths: Cinematic, portfolio-quality presentation. Large visual areas perfect for showing actual product screenshots or brand imagery. Asymmetric grid creates visual interest. Text overlays keep information accessible while prioritizing visuals. Immediately shows breadth of work.

Visual Treatment: In production, each tile would feature actual project screenshots, brand colors, or product visuals. Gradient overlays ensure text readability. Floating blur orbs add depth and atmosphere. Scale effect on hover creates tactile feedback.

Tradeoffs: Requires high-quality images for each case study. Less text information visible at a glance. Grid layout may not work as well for fewer than 3 case studies. More complex responsive behavior.
