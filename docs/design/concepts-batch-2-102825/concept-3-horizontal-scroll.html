Concept 3: Horizontal Scroll Carousel

Strategy: Horizontal scrolling inspired by streaming services (Netflix, Spotify). Wide cards show more detail for each case study. First card centers on load, creating immediate focus.

Strengths: Immersive browsing experience that invites exploration. Wider cards allow for more visual space and richer descriptions. Feels modern and app-like. Natural on touch devices. Scales to any number of case studies without layout changes. Visually distinctive from typical portfolio layouts.

Interaction Details: Drag to scroll (grab cursor). Smooth scrolling with momentum. Custom purple scrollbar as visual accent. Scale effect on hover. Arrow indicator slides in. Grid pattern in visual area adds subtle texture without distraction.

Implementation: CSS scroll-snap could be added for card alignment. JavaScript could enhance with keyboard navigation (arrow keys) and auto-scroll hints. Scrollbar color matches brand purple.

Tradeoffs: Horizontal scrolling is less familiar than vertical. Not all three case studies visible at once (requires interaction to see all). Some users may not discover the scroll affordance. More complex to implement with proper accessibility (keyboard navigation, screen readers).
