Concept 1: Minimal Vertical Cards

Strategy: Large typography and generous spacing create a premium, editorial feel. Vertical stacking eliminates the cognitive load of comparing across columns — users naturally read top to bottom.

Strengths: Scalable to any number of case studies without layout changes. Hover interactions reveal depth gradually. Works beautifully on mobile with minimal adaptation. Large type sizes establish visual hierarchy immediately.

Interaction Details: Purple accent bar slides in on hover. Title gains gradient on hover. Description fades in smoothly. Metric becomes fully opaque. All transitions use cubic-bezier easing for smooth, premium feel.

Tradeoffs: Requires more scrolling to see all three case studies. Less visual/cinematic than image-first approaches. Relies on strong typography and copy rather than visuals.
