Minimal Variant B: Offset Image
Strategy: Places a vertical portrait-oriented image to the right side of the content, creating visual interest through asymmetry while maintaining clean, minimal layout. The image floats alongside the text rather than being stacked, creating a more dynamic composition without sacrificing the minimal aesthetic.

Strengths: Creates visual interest through layout asymmetry, image feels integrated not appended, portrait orientation is distinctive, maintains minimal clean feel, responsive layout stacks naturally on mobile
