Design Direction

This mood board explores visual directions for two key interaction patterns: inline progressive reveal for AI projects and split screen journey for case studies. The goal is to create engaging, modern interfaces that reduce page transitions while providing rich, detailed content exploration.

Inspiration & Rationale

These design choices draw from modern web aesthetics while avoiding overused AI-generated patterns.

Why These Choices?

Dark-first design: The existing portfolio is dark mode, so we embrace it fully with deep blacks and vibrant accents that glow.

Distinctive typography: Crimson Pro (serif elegance), JetBrains Mono (technical precision), Manrope (modern geometric) - each creates a different personality and avoids the Inter/Roboto trap.

Progressive disclosure: Inline expansion reduces navigation friction and keeps users in flow while exploring AI projects.

Split screen navigation: Case studies are stories - the split screen creates a narrative journey with clear progress and structure.

Three distinct concepts: Not just color variations - each concept explores different information architecture, interaction models, and visual aesthetics.
